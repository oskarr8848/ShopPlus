Built For Survival Economies

ShopPlus keeps the common economy flows in one plugin while staying compatible with the stacks servers already use.

GUI Shops

Server shop categories, player market browsing, sell menus, and worth lists.

Player Listings

Players can list held items, manage active sales, and buy from others without unsafe trades.

Secure Trades

Trade request, accept, deny, and confirmation flows designed for busy survival servers.

Vote Rewards

Reward hooks for money, Merra points, temporary discounts, ranks, and flight-time systems.

Merra Currency

Global points via Merra API, account linking, and PlaceholderAPI value display.

Admin Reports

Market reports, price reset tools, and reload commands for live server management.

Merra

Merra support is designed around an API-first model. Public servers never receive direct database credentials.

Server plugin -> https://api.merra.network -> Merra API -> Merra database

The public placeholder exposed by ShopPlus is %mcnp_merra_points%.
